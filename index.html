// THE EVENT

Build. Break. Innovate.

The CodeWIT Code-A-Thon is a fast-paced, individual coding challenge designed to bring together students of all skill levels for an exciting test of logic, creativity, and problem-solving. Inspired by HackerRank-style questions, the event features a curated set of problems tailored to the programming language each participant chooses. All challenges are equal in difficulty and can be completed within the event's time limit, ensuring a fair and engaging experience for everyone.

Whether you're a beginner looking to build confidence or an experienced programmer ready to compete, the CodeWIT Code-A-Thon welcomes you.
